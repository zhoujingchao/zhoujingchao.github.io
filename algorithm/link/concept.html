# 概念

链表是一种物理存储单元上非连续、非顺序的存储结构，数据元素的逻辑顺序是通过链表中的指针链接次序实现的。一个对象存储着本身的值和下一个元素的地址。

链表的特性是在中间任意位置添加删除元素的都非常的快，不需要移动其它的元素。通常链表每一个元素都要保存一个指向下一个元素的指针（单链表），增加和删除一个元素对于链表数据结构就非常简单了，只要修改元素中的指针就可以了。
